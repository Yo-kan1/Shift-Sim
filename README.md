# Shift-Sim (バイト収入シミュレーター)

## 概要
「今月のシフトでいくら稼げるか」を事前にシミュレーションし、目標金額の達成や扶養範囲内（103万・130万の壁など）での調整をサポートするWebアプリケーションです。

## 主な機能（予定）
- 複数のバイト先の時給・交通費・深夜手当などの条件登録
- シフトの予定入力と、リアルタイムな予想収入の計算
- 目標金額や扶養上限までの「残り可能労働時間・金額」の可視化

## 技術スタック
- **Backend:** Python (FastAPI), SQLModel
- **Database:** SQLite (開発用) / PostgreSQL (本番想定)
- **Frontend:** HTML / CSS / JavaScript
